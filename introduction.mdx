# Welcome to Maestro

> Maestro hones fractured attention into focused intent.

Maestro is a cross-platform desktop app for orchestrating your fleet of AI agents and projects. It's a high-velocity solution for hackers who are juggling multiple projects in parallel. Designed for power users who live on the keyboard and rarely touch the mouse.

## How It Works

Maestro is a **pass-through to your AI provider**. Whatever MCP tools, skills, permissions, or authentication you have configured in Claude Code, Codex, or OpenCode works identically in Maestro. The only difference is we're not running interactively—each task gets a prompt and returns a response, whether it's a new session or resuming a prior one.

<Note>
Collaborate with AI to create detailed specification documents, then let Auto Run execute them automatically, each task in a fresh session with clean context. Allowing for long-running unattended sessions, with continuous runtime records approaching 24 hours.
</Note>

## Run Multiple Agents in Parallel

Run multiple agents in parallel with a Linear/Superhuman-level responsive interface. Currently supporting:

- **Claude Code** - Anthropic's AI coding assistant (fully integrated)
- **OpenAI Codex** - OpenAI's coding agent (fully integrated)
- **OpenCode** - Open-source AI coding assistant (fully integrated)
- **Factory Droid** - Factory's AI coding assistant (fully integrated)

Plans for additional agentic coding tools (Gemini CLI, Qwen3 Coder) based on user demand.

## Key Features at a Glance

### Power User Features

- **Dual-Mode Sessions** - Each agent has both an AI Terminal and Command Terminal (`Cmd+J` to switch)
- **Quick Actions** - Press `Cmd+K` / `Ctrl+K` to access any feature instantly
- **Session Discovery** - Automatically discovers and imports existing sessions from all supported providers
- **Git Integration** - Automatic repo detection, branch display, diff viewer, commit logs
- **File Explorer** - Browse project files with syntax highlighting, markdown preview, and image viewing
- **Message Queueing** - Queue messages while AI is busy; they're sent automatically when ready
- **Draft Auto-Save** - Never lose work. Drafts are automatically saved and restored per session
- **Cost Tracking** - Real-time token usage and cost tracking per session and globally
- **Usage Dashboard** - Comprehensive analytics for tracking AI usage patterns (`Opt+Cmd+U` / `Alt+Ctrl+U`)

### Analytics & Visualization

- **Usage Dashboard** - Track AI usage patterns with aggregated statistics, activity heatmaps, and CSV export
- **Document Graph** - Visual knowledge graph of your markdown documentation with interactive navigation
- **Director's Notes** - Bird's-eye view of all agent activity in a unified timeline with AI-powered synopses
